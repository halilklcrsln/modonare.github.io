MODONARE ASTRO-ACOUSTIC HERO IMAGES AND SCREENSHOTS DIRECTORY
--------------------------------------------------------------

This directory contains key screenshots and artwork for the immersive fanned mockups
located on the left and center of the hero section.

To replace the beautiful abstract fallback vector designs with your actual application
screenshots, simply drop your files right here inside this folder using the following exact names:

1. Fanned Phone 1 (Back-most screen) -> hero_screen_1.png
2. Fanned Phone 2 (Middle screen)    -> hero_screen_2.png
3. Fanned Phone 3 (Front-most screen) -> hero_screen_3.png

Active Mode cover arts inside the center phone mockup:
- Relax Cover Art      -> relax_art.png
- Focus Cover Art      -> focus_art.png
- Motivation Cover Art -> motivation_art.png

When you upload or save files with these exact names, the application will pick them up
automatically and overlay them inside the high-fidelity mockups!
